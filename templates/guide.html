How To Use HR Guru ATS

Use this guide as an operating checklist. Follow the steps in order when creating a requirement, adding candidates, matching CVs to a JD, and exporting recruiter-ready summaries.

Quick Start

  1. Create or open a requirement.
  2. Attach the latest JD file to that requirement.
  3. Add candidates manually or through bulk upload.
  4. Use JD Match to compare one JD against multiple CVs.
  5. Open the ranked candidate result and review strengths, concerns, missing skills, role fit, domain fit, and summary.
  6. Export the match PDF when sharing with the hiring manager.

1. Create And Maintain Requirements

  1. Open Requirements from the left menu.
  2. Click Add Requirement.
  3. Enter requirement title, client, assigned sourcer, assigned recruiter, daily target, and status.
  4. Attach the JD using Attach Job Description.
  5. Click Save Requirement.
If the JD changes later, open the requirement with Edit, upload the new JD, and save. The latest uploaded JD becomes the tagged JD for that requirement.

2. Add Candidates

  1. Open Candidates.
  2. Click Add Candidate.
  3. Select the requirement first.
  4. Upload the CV and click parse if available.
  5. Review extracted fields: name, email, phone, company, role, experience, skills, notice period, salary, and location.
  6. Complete any missing mandatory fields.
  7. Submit the candidate.
Non-admin users can see only candidates uploaded or assigned to them. Admin users can see all candidates.

3. Use JD/CV Match

Match Multiple Candidates Against One JD

  1. Open JD Match.
  2. Select Candidate Matching.
  3. Paste the JD text or upload a JD file.
  4. Select one or more candidate CV files.
  5. Optionally enter Custom Screening Rules, such as Must have Kubernetes or Required: Bangalore location.
  6. Click Analyze.
  7. Review the right-side ranking list. Candidates are sorted by match score.
  8. Click a candidate name in the ranking list to open detailed match results.

Export Match Results

  1. Open the selected candidate's match dashboard.
  2. In Hiring Manager Summary, click Export PDF.
  3. The exported file includes the candidate name in the filename.

4. Generate Interview Screening Questions

  1. Open JD Match.
  2. Select Interview Questions.
  3. Paste the JD text or upload the JD file.
  4. Click Generate Questions.
  5. Questions open in a separate popup window.
  6. Click Export PDF in the popup to download the screening questions.
If Ollama is running, the app uses the LLM for richer questions. If Ollama is unavailable, it generates deterministic fallback questions from mandatory JD skills.

5. Review And Manage Candidates

  • Use search to find by name, role, company, skill, email, or phone.
  • Use client and requirement filters to narrow the list.
  • Update candidate status after each action.
  • Open a candidate to view or edit details.
  • Upload a replacement CV from the candidate edit modal when needed.

6. Use Reports

  1. Open Reports.
  2. Select report type: summary, sourcer, status, skills, submissions, or daily dashboard.
  3. Set date range or use quick filters like Today, This Week, or This Month.
  4. Click Generate.
  5. Export reports when needed.

7. Admin Tasks

User Management

  1. Open User Management from the left menu.
  2. Add users or create logins from team members.
  3. Assign Admin or Bulk Upload access as needed.
  4. Reset passwords for users when required.
  5. Approve new user registration requests from the approval email link.
